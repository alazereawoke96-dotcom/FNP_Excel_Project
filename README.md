# FNP_Excel_Project

🚀 Project Overview
This project presents an interactive Sales Analysis Dashboard built using Microsoft Excel to analyze business performance, uncover trends, and generate actionable insights.  The dashboard transforms raw data into meaningful visualizations, enabling better decision-making across sales, marketing, and operations.

📌 Key Highlights
📦 1,000 Orders Analyzed
💰 ₹3,520,984 Total Revenue
🛒 ₹3,520.98 Avg Customer Spending
🚚 5.53 Days Avg Delivery Time
📸 Dashboard Preview

📊 Features
🔹 Revenue analysis by Occasion and Category
🔹 Top 10 Cities by order volume
🔹 Monthly sales trend analysis 📅
🔹 Top 5 revenue-generating products 🛍️
🔹 Hourly order trends ⏰
🔹 Interactive Slicers & Filters

🧠 Key Insights
🎯 Anniversary & Raksha Bandhan drive the highest revenue
🎨 Colors category contributes the most to overall sales
📅 Peak sales months: February & August
🌆 Certain cities consistently outperform others
⏰ Orders peak during afternoon & evening hours

🛠️ Tools & Techniques
Microsoft Excel
Pivot Tables & Pivot Charts
Data Cleaning & Transformation
Slicers & Interactive Dashboarding
Data Visualization Best Practices
💡 Business Value

This dashboard helps:
Identify high-performing products & occasions
Understand customer purchasing behavior
Optimize marketing strategies
Improve delivery and operational efficiency

▶️ How to Use
Download the Excel file
Open in Microsoft Excel
Go to Dashboard Sheet
